# Taller: API

* Introducción: APIs de alto rendimiento.
* Diseño de protocolos de seguridad para API.
* Manejo de sesiones en APIs de alto rendimiento.

### Requisitos

* Nodejs (7+) & NPM
* MongoDB
* Redis

```
> mkdir taller
> cd !$ && git clone git@github.com:aichholzer/taller.git
> npm install
```

### Introducción

```
> cd introduccion
> node app/
```

### Protocolos de seguridad

```
> cd protocolos
> node app/
```

### Sesiones

```
> cd sesiones
> node app/
```
